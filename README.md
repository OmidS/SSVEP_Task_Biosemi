*****************
**SSVEP_OpenBCI**
*****************


Requirements:

	Python 3


	Psychopy v3.1.4
		download or install dependencies

	OpenBCI Board v2


To start:

	in console:
		python run_ssvep.py

	in Psychopy Coder:
		open run_ssvep.py
		hit CTRL + r

	Enter Participants Name and Session Number in GUI
