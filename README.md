Requirements:

	Python 2.7


	Psychopy v1.88
		download or install dependencies

	OpenBCI Board v2


To start:

	in console:
		python run.py

	in Psychopy Coder:
		open run.py
		hit CTRL + r

	Enter Participants Name and Session Number.
